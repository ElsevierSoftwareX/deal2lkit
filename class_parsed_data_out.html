void ParsedDataOut< dim, spacedim >::write_data_and_clear ( const Mapping< dim, spacedim > &  mapping = StaticMappingQ1<dim,spacedim>::mapping)

Actually write the file.

Once the data_out has been prepared, vectors have been added, the data can be written to a file. This is done in this class. At the end of this function call, data_out and output_file are in a pristine situation, and the process can be started again. used_files is an optional variable that takes a list of useful files (ex. "parameter.prm time.dat") and copies these files in the incremental_run_prefix of the costructor function.
