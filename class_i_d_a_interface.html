Detailed Description

template<typename VEC = Vector<double>>
class IDAInterface< VEC >

Interface to SUNDIALS IDA library.

deal2lkit features an interface for the SUite of Nonlinear and DIfferential/ALgebraic equation Solvers (SUNDIALS). Each of the SUNDIALS solvers require the user to derive its class from the base class SundialsInterface, providing function evaluation, jacobian evaluation, etc. to the underlying SUNDIALS solver.

The class IDAInterface is a wrapper to the Implicit Differential-Algebraic solver which is a general purpose solver for systems of Differential-Algebraic Equations (DAEs).
