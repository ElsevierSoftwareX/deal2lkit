Detailed Description

template<int dim>
class ParsedQuadrature< dim >

A deal2lkit wrapper for an iterated version of dealii::QuadratureSelector.

The template integer specifies the dimension of the quadrature.

Constructor: takes an optional name for the section.

Moreover, it Takes the name of the quadrature rule quadrature_type that could be chosen among following names:

  • gauss;
  • midpoint;
  • milne;
  • simpson;
  • trapez;
  • weddle; If quadrature_type is "gauss" it is possible to specify the number of quadrature points in each coordinate direction (order).

The last parameter specify how many times this quadrature should be repeated, i.e., in one space dimension, the given base formula is copied and scaled onto a given number of subintervals of length 1/repetitions. In more than one space dimension, the resulting quadrature formula is constructed by tensor product of the respective iterated quadrature formula in one space dimension.
